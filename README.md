# Rebuilt Voices

## تشغيل المشروع

1. إعداد قاعدة بيانات PostgreSQL باسم rebuilt_voices.
2. تعديل بيانات الاتصال في backend/models/slide.js (user, password, host).
3. تثبيت حزم backend:
   npm install
4. تشغيل السيرفر:
   npm start
5. تثبيت حزم frontend:
   cd frontend
   npm install
6. تشغيل الواجهة الأمامية:
   npm start
7. زيارة http://localhost:3000
